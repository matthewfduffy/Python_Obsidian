#keyword 
## not 

Logical Operator (along with [[and]], [[or]])

(x and y) --> both x and y must be True
(x or y) --> either x or y must be True
(not x) --> x must be false

```
x, y = True, False
(x or y) == True
(x and y) == False
(not y) == True
```

Related: [[and]], [[or]]
