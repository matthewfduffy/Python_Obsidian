#file-method 	
### .isatty()
The `isatty()` method returns `True` if the file stream is interactive (e.g. - connected to a terminal device)

##### Syntax:
`file.isatty()`

##### Example(s):
```python
f = open("demofile.txt", "r")
print(f.isatty)
```
