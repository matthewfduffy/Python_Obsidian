#list-method 
### .count()
Returns the number of elements with the specified value.

##### Syntax:
 `list.count(value)`

##### Parameter Value(s):
| Parameter | Description                                                                     |
| --------- | ------------------------------------------------------------------------------- |
| value     | Required. Any type (string, number, list, tuple, etc). The value to search for. | 


##### Examples:
```python
points = [1, 4, 2, 9, 7, 8, 9, 3, 1]  
  
x = points.count(9)
print(x)	# 2
```
