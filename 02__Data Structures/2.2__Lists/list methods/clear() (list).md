#list-method 
### .clear()
The `clear()` method removes all the elements from a list.

##### Syntax:
`list.clear()`

##### Parameter Values:
None.

##### Examples:
```python
fruits = ['apple', 'banana', 'cherry', 'orange']  
  
fruits.clear()
  
print(fruits)
#
```
