#list-method 
### .reverse()
The `reverse()` method reverses the sorting order of the elements.

##### Syntax:
 `list.reverse()`

##### Parameter Values:
None                                                                                                


##### Examples:
```py
fruits = ['apple', 'banana', 'cherry']  
  
fruits.reverse()
print(fruits)	
# ['cherry', 'banana', 'apple']
```
