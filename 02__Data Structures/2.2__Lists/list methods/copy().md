#list-method 
### .copy()
The `copy()` method returns a copy of the specified list.

##### Syntax:
`list.copy()`

##### Parameter Values:
None.

##### Examples:
```py
fruits = ['apple', 'banana', 'cherry', 'orange']  
  
x = fruits.copy()
  
print(x)
```
