#set-method
## add()
The `add()` method adds an element to the set.

If the element already exists, the `add()` method does not add the element.

##### Syntax:
`set.add(elmnt)`

##### Parameter(s):
| Parameter | Desc.                                    |
| --------- | ---------------------------------------- |
| elmnt     | Required. The element to add to the set. |


##### Example(s):
```python
fruits = {"apple", "banana", "cherry"}  
  
fruits.add("orange")  
  
print(fruits)
```
