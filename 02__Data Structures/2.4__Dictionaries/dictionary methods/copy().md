#dictionary-method
## .copy()
The `copy()` method returns a copy of the specified dictionary


##### Syntax:
 `dictionary.copy()`

##### Parameter Values:
None.


##### Examples:
```python
car = {  
  "brand": "Ford",  
  "model": "Mustang",  
  "year": 1964  
}  
  
x = car.copy()  
print(x)

```
